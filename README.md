# Rafif Ahmad Portfolio

Personal portfolio built with React, Vite, Tailwind CSS, Framer Motion, and Lenis.

## Scripts

- `npm run dev` — start the local development server
- `npm run build` — create the production build
- `npm run lint` — run ESLint
- `npm run preview` — preview the production build locally

## Project structure

- `src/` — app source, sections, shared UI, and theme context
- `public/` — static images and downloadable assets

## Notes

- The portfolio uses shared section wrappers to keep spacing and animation behavior consistent.
- `public/resume.pdf` is the downloadable CV linked from the contact section.
